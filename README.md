## Getting Started

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/imgagandeep/search.git
   cd search
   ```

2. Install dependencies
   
   ```
   npm install
   ```

3. Start the development server
   
   ```
   npm run dev
   ```

4. Go to URL
   
   ```
   http://localhost:5173/
   ```
